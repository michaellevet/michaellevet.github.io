    • 2.2: For a function f(x), the derivative f'(x) provides the rate of change. At a point c, if f'(c) > 0, we say that f is increasing at c. If f'(c) < 0, then f is decreasing at c. If f'(c) = 0, then f is constant at c. You are welcome to use this fact on quizzes and tests. Remember that we f'(x) is obtained by simplifying the difference quotient [f(x+h)-f(x)]/h, then plugging in h = 0.
